# 🔍 GitHub User Data Fetcher

A lightweight, responsive web application that allows users to search for any GitHub profile and instantly fetch their public data using the GitHub REST API. Built purely with HTML, CSS, and vanilla JavaScript.

## 🚀 Features

* **Real-time Search:** Look up any active GitHub username.
* **Profile Overview:** Displays user avatar, name, bio, and account creation date.
* **Repository Statistics:** Shows the total number of public repositories, followers, and following count.
* **Direct Links:** Quick access buttons to view the user's actual GitHub profile.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3
* **Logic & API Handling:** Vanilla JavaScript (ES6+)
* **API Used:** [GitHub REST API](https://docs.github.com/en/rest)
